# **App Name**: Neon SaaS Landing

## Core Features:

- Hero Section: A visually striking hero section with a gradient background and CTA button to attract users.
- Feature Cards: Interactive feature cards that showcase the key benefits of the SaaS product.
- Workflow UI: A step-by-step workflow section with animated transitions to illustrate the user journey.
- Pricing Tiers: Professionally designed pricing cards with A/B testing toggle to highlight different subscription options.
- Clean Footer: Minimal and clean footer with dark mode support and essential links for improved navigation.
- Axios API Wrapper: API wrapper using Axios for making HTTP requests to the backend with placeholder endpoints.

## Style Guidelines:

- Primary color: Electric blue (#7DF9FF) to create a modern, vibrant, SaaS aesthetic.
- Background color: Light bluish-gray (#E5E7EB), slightly desaturated to focus attention on primary elements.
- Accent color: Deep purple (#6A0DAD) to complement the electric blue.
- Body and headline font: 'Inter' (sans-serif) for a modern and readable design.
- Use Lucide icons for consistent and clean visual elements across the entire landing page.
- Grid layout for sections, ensuring a responsive design that looks premium on both desktop and mobile devices.
- Framer Motion for fade-in effects, hover scale animations, and staggered transitions, providing a smooth user experience.
